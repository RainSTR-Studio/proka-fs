# Proka FS Documentation

This file contains the summary and the index of the proka fs.

Also, this will provide some APIs, and you can use them after you read this.

# Summary

This project (`proka-fs`) is the file system, which will be used in `proka-kernel`, our 
main kernel project.

We hope it's lightweight, and it can be used in embedded systems.
